## About PHP-46Elks

PHP-46Elks is a wrapper for [46elks.se](46elks.se) written for PHP

### Installation

*General PHP use*
```
composer require tarre/php-46elks
```

*For Laravel (Includes notification driver)*
```
composer require tarre/laravel-46elks
```
